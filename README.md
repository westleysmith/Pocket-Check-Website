# Pocket-Check-Website

Static marketing and policy site for the Pocket Check app. Hosted via GitHub Pages at `pocketcheck.app`.

## Structure

- `index.html` - home / features
- `pricing.html` - free vs Pro tiers
- `support.html`, `retirement-planner.html` - tools and help
- `privacy-policy.html`, `terms-of-service.html`, `data-retention-policy.html`, `information-security-policy.html` - legal
- `plaid-oauth.html` - Plaid OAuth redirect page
- `css/styles.css` - brand tokens (navy `#0f1f2e`, green `#2ecc71`) and all page styling
- `js/retirement-planner.js` - retirement calculator logic

## Brand Assets

- `assets/logo.svg` - green gradient pocket mark, transparent background (used in nav on every page)
- `favicon/` - full favicon set (16-512px PNG, `.ico`, Apple touch icon)
