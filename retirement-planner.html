Advanced settings

These override the basic retirement inputs when enabled. If you're early in your career and not sure about tax bracket details, the basic mode is a reasonable approximation.

Return outlook

How optimistic should the simulation be about future returns? The Monte Carlo still explores 10,000 random paths - this picks the mean of the distribution. Three numbers below are stock / bond / cash real return assumptions.

Taxes

Replaces the basic "effective tax rate" with 2026 federal brackets plus standard deduction. Ordinary income brackets only (no separate LTCG / qualified-dividends rates yet).

Portfolio fees

Subtracted from each year's return. A typical index fund is 0.03-0.15%; a managed fund is 0.5-1.0%; an advisor adds another 0.5-1.5%.

Social Security claim strategy

Your benefit at full retirement age (67). Claiming earlier reduces it, waiting until 70 increases it. Leave 0 to use the basic "Social Security (annual)" input as-is.

Required Minimum Distributions

Starting at age 73, the IRS requires you to withdraw a fraction of your tax-deferred (401k/IRA) balance each year whether you need the money or not. Advanced mode applies this automatically using the IRS Uniform Lifetime Table.

Career stages

One row per phase of your career. Salary is flat within a stage. For a raise or role change, add another row at that age. Rows are auto-sorted by age when you run the simulation.

Assumptions and limitations
  • All figures are in real (today's) dollars. Inflation is not added back.
  • Real return assumptions (annual): stocks 6.8% / σ=17%, bonds 2.0% / σ=6%, cash 0.3% / σ=1%. Stock-bond correlation 0.10.
  • Withdrawal order in retirement: taxable, then tax-deferred (grossed up for taxes), then tax-free.
  • Taxes: a single effective rate is applied to tax-deferred withdrawals only. No bracket logic, RMDs, or state tax.
  • Contribution routing: 70% of your savings into tax-deferred, 30% into taxable. Employer match all goes tax-deferred.
  • Salary is flat within a stage. To model a raise, add another stage.
  • Allocation: same across all account types. Linear glide path from now to retirement age.
  • Social Security: fixed real annual benefit starting at claim age.
  • Not modeled: health shocks, variable spending, Roth conversions, RMD-forced withdrawals.

This tool is for educational planning only and is not financial advice. Talk to a licensed advisor before making major financial decisions.
